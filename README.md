# cj 오쇼핑 코딩테스트

## 개발 환경 스팩

* Build Tool : maven
* Framework : Springboot + Spring web starter 
* JVM : java8
* Database : h2
* Persistence framework : mybatis

## 소스 구현 사항

* 게시글 목록
* 게시글 등록
